# MathsLoop

A black-and-white smooth React/Vite maths Q&A prototype.

## Deploy on Vercel

1. Upload this folder to GitHub.
2. Import the GitHub repo into Vercel.
3. Use:
   - Framework: Vite
   - Build command: `npm run build`
   - Output directory: `dist`

## Note

This is currently a front-end demo. Questions, answers and notifications are stored in browser `localStorage`, so they are not shared between different users/devices until a database is added.
